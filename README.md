# 📚 AI Study Assistant

## 📖 Overview

AI Study Assistant is an AI-powered web application that helps students learn more effectively. Users can paste their study notes to generate AI summaries, exam questions, flashcards, and ask questions based on the notes. The application also logs user activity to Google Sheets using Make.com automation.

## ✨ Features

* 📝 Generate AI-powered summaries
* 📚 Choose difficulty levels (Easy, Medium, Hard)
* ❓ Generate exam questions with answers
* 🗂 Generate flashcards
* 💬 Ask questions about study notes
* 📊 Save activity to Google Sheets using Make.com
* ☁️ Deployed online using Render

* `app.py` - Flask backend
* `templates/` - HTML files
* `static/` - CSS and JavaScript files
* `requirements.txt` - Python dependencies

## ▶️ How to Run

1. Clone the repository.
2. Install dependencies:

   ```
   pip install -r requirements.txt
   ```
3. Create a `.env` file and add:

   ```
   OPENROUTER_API_KEY=your_api_key
   ```
4. Run the application:

   ```
   python app.py
   ```
5. Open:

   ```
   http://127.0.0.1:5000
   ```
